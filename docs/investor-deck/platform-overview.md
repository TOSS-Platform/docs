# Platform Overview

## TOSS Platform Components

### For Fund Managers

- **Fund Creation Wizard**: Launch fund in less than 5 minutes
- **Trading Interface**: Execute trades with RiskEngine validation
- **Performance Dashboard**: Real-time NAV, returns, risk metrics
- **Investor Management**: View/manage investor base
- **Fee Management**: Automatic fee calculation and distribution

### For Investors

- **Fund Discovery**: Browse and filter funds by strategy, risk, performance
- **Portfolio Tracking**: Real-time positions and performance
- **Deposit/Withdrawal**: Instant on zkSync L2
- **Performance Analytics**: Historical returns, risk metrics
- **Governance**: Vote on protocol parameters

### For the Protocol

- **RiskEngine**: Validates every trade in real-time
- **NAV Engine**: Calculates fund valuations hourly
- **SlashingEngine**: Enforces violations automatically
- **DAO Governance**: Decentralized parameter management
