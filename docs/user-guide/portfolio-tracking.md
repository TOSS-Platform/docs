# Portfolio Tracking

## Overview

Guide to tracking and monitoring your portfolio performance in TOSS.

## Viewing Your Portfolio

### Dashboard Overview

Access your portfolio dashboard to see:
- Total portfolio value
- Fund allocation breakdown
- Performance metrics
- Recent activity

### Fund-Level Details

For each fund, you can view:
- Current NAV and share price
- Holdings and asset allocation
- Performance over time
- Risk metrics
- Fee breakdown

## Performance Metrics

### Key Metrics Explained

- **NAV per Share**: Current value of each fund share
- **Total Return**: Percentage gain/loss since investment
- **High Water Mark**: Highest NAV achieved (used for performance fees)
- **Drawdown**: Current decline from peak

### Performance Charts

View performance over:
- Last 24 hours
- Last 7 days
- Last 30 days
- Last 90 days
- Last year
- All time

## Asset Allocation

### Current Holdings

View your holdings:
- Asset balances
- USD values
- Percentage of NAV
- Concentration metrics

### Allocation Analysis

Understand your portfolio:
- Diversification level
- Risk distribution
- Asset class breakdown

## Real-Time Updates

- NAV updates hourly (or more frequently during high volatility)
- Trade updates in real-time
- Performance metrics recalculated automatically
