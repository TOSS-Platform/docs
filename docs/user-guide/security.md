# Security Best Practices

## Overview

Security guidelines for protecting your TOSS funds and account.

## Wallet Security

### Private Key Management

- **Never Share**: Never share your private key or seed phrase
- **Use Hardware Wallets**: Consider hardware wallets for large holdings
- **Backup Securely**: Store backups in secure, offline locations
- **Multi-Signature**: Use multi-sig for additional security

### Session Keys

For Fund Managers:
- Use session keys for trading
- Limit session key permissions
- Set expiration times
- Rotate regularly

## Account Security

### Access Control

- **Strong Passwords**: Use unique, strong passwords
- **2FA**: Enable two-factor authentication where available
- **Regular Reviews**: Review account access regularly
- **Revoke Access**: Remove unused permissions

### Monitoring

- Monitor account activity regularly
- Review all transactions
- Set up alerts for unusual activity
- Report suspicious activity immediately

## Fund Security

### Risk Limits

- Set appropriate risk limits
- Monitor position sizes
- Avoid over-concentration
- Respect drawdown limits

### Emergency Procedures

- Know emergency withdrawal procedures
- Understand fund closure process
- Have backup plans ready
