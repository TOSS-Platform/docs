# AI Prompt Examples

## Overview

Example prompts you can use with AI assistants connected to TOSS MCP to interact with the protocol naturally.

## Fund Management Prompts

### Creating a Fund

```
"Create a balanced fund with $2M projected AUM. Set management fee to 2% and performance fee to 15%."
```

```
"I want to start an alpha fund with 5x leverage limit. What's the minimum stake required?"
```

### Querying Fund Information

```
"Show me details of fund ID 42, including current NAV, holdings, and performance."
```

```
"What are the risk parameters for my fund?"
```

## Trading Prompts

### Executing Trades

```
"Execute a trade: swap $10,000 USDC for BTC with maximum 2% slippage."
```

```
"Can I increase my BTC position by $50,000? Check if it violates any risk limits."
```

### Trade Analysis

```
"Show me all trades executed in fund 42 today."
```

```
"Calculate the impact of a $100k BTC purchase on my portfolio concentration."
```

## Investor Operations Prompts

### Deposits

```
"I want to deposit $50,000 into fund 42. How many shares will I receive?"
```

```
"Check if I'm eligible to invest in fund 42 based on my investor class."
```

### Withdrawals

```
"Request withdrawal of 10,000 shares from fund 42."
```

```
"When can I withdraw from fund 42? Check lockup period."
```

## Best Practices

1. **Be Specific**: Include fund IDs, amounts, and timeframes
2. **Check First**: Use "check" or "validate" prompts before executing
3. **Ask for Details**: Request explanations of calculations or decisions
4. **Safety First**: AI will warn you of risks - listen to warnings
