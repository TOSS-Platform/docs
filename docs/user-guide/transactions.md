# Transaction Management

## Overview

Guide to managing transactions in TOSS: deposits, withdrawals, trades, and tracking.

## Deposits

### Making a Deposit

1. Navigate to fund page
2. Click "Deposit"
3. Enter amount
4. Approve asset transfer
5. Confirm deposit

### Deposit Processing

- Shares calculated based on current NAV
- Shares minted immediately
- Deposit reflected in portfolio

## Withdrawals

### Requesting a Withdrawal

1. Navigate to fund page
2. Click "Withdraw"
3. Enter share amount or USD amount
4. Review fees
5. Submit withdrawal request

### Withdrawal Processing

- Request added to queue
- Processed after queue delay (24 hours)
- Assets transferred to your wallet

## Trade History

### Viewing Transactions

Access transaction history to see:
- All deposits
- All withdrawals
- All trades (if you're Fund Manager)
- Transaction status
- Timestamps and amounts

### Transaction Details

Each transaction shows:
- Transaction ID
- Type (deposit, withdrawal, trade)
- Amounts and assets
- Status (pending, completed, failed)
- Timestamp
- Block number (on-chain transactions)
