# Investor Onboarding Process

## Overview

New investor registration and initial setup.

## Steps

1. **Connect Wallet**: Web3 wallet connection
2. **Auto-Registration**: First deposit triggers registration
3. **Class Assignment**: Starts as RETAIL
4. **TOSS Staking** (optional): Stake for class upgrade

## Initial State

```yaml
Class: RETAIL
ICS Score: 0 (calculated after first activity)
State: ACTIVE
Access: Tier 1-2 funds only
```

## Upgrade Path

```
RETAIL (100 TOSS) → PREMIUM (1k TOSS) → INSTITUTIONAL (10k TOSS) → STRATEGIC (100k TOSS)
```
